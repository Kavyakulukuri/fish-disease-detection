# 🐟 Fish Disease Detection using Deep Learning (VGG16)

## 📌 Overview

This project is an AI-powered application that detects diseases in fish using image classification. It uses a Convolutional Neural Network (CNN) with the VGG16 architecture to identify fish diseases from images and provide quick results.

The system helps fish farmers and researchers detect diseases early, reducing loss and improving fish health management.

---

## 🚀 Features

* 🧠 Deep Learning model using VGG16
* 📷 Image-based disease detection
* ⚡ Fast and accurate predictions
* 🖥️ Simple and user-friendly interface
* 📊 Real-world application in aquaculture

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Kavyakulukuri/fish-disease-detection.git
cd fish-disease-detection
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```
